

## Enhanced Project Card PDF Export

**Risk: GREEN** -- purely UI/PDF generation changes, no DB/auth/RLS modifications.

### What's Missing Today

The current Project Card PDF contains:
- Project header (name, status, dates, metadata)
- Description
- Assigned Personnel table (Name, Role, Location, Email, Phone)
- Calendar Items table (Date, Description, Milestone)

It does NOT include:
- Project Phases
- Personnel compliance status (certificate validity)
- A mini competence matrix
- Milestones and Events separated clearly

### What the Enhanced Project Card Will Include

1. **Project Info** (unchanged) -- Name, status, duration, project number, customer, location, project manager, work category, description

2. **Project Phases** (NEW) -- Table with phase name, start date, end date, and duration

3. **Milestones** (IMPROVED) -- Separated from events into its own table, sorted by date

4. **Events** (IMPROVED) -- Non-milestone calendar items in their own table, sorted by date

5. **Assigned Personnel with Compliance** (IMPROVED) -- Add a compliance status column (Compliant / Expiring / Non-Compliant) per person based on their certificate statuses

6. **Personnel Competence Matrix** (NEW) -- An inline mini-matrix showing each person's certificates with V/E/X status indicators, similar to what the standalone Competence Matrix PDF has, but embedded in the Project Card for a single comprehensive document
