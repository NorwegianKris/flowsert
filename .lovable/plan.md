

# Add Certificate Bundle PDF Export (Single-Person v1)

## Summary
Create a new "Generate Certificate Bundle" export option in both External Sharing and Project Sharing dialogs. This merges all uploaded certificate documents for a single selected person into one audit-ready PDF with cover page and separator pages.

## Changes

### 1. New file: `src/lib/mergeCertificatesPdf.ts`

Core utility with `generateCertificateBundlePdf()` function.

**Interface:**
```ts
export interface CertificateBundleOptions {
  person: Personnel;
  companyName?: string;
  projectName?: string;
  onProgress?: (current: number, total: number, label: string) => void;
}

export interface CertificateBundleResult {
  doc: jsPDF;
  included: number;
  skipped: number;
}
```

**Logic flow:**
1. Filter person's certificates to only those with a `documentUrl` (sort alphabetically by name)
2. Pre-flight safety check: if certificates with documents > 30, throw with descriptive message (caller handles confirmation)
3. Create jsPDF (portrait A4)
4. **Cover page:**
   - Centered title: `CERTIFICATE BUNDLE` (font 16, bold)
   - Subtitle: `FlowSert Workforce Compliance` (font 9, grey)
   - Metadata block (left, muted grey, font 8): Company, Project (if applicable), Person name, Generated timestamp
   - Thin divider line
5. **For each certificate (sequentially):**
   - Call `onProgress(current, total, certName)`
   - Add separator page with certificate metadata: name, issuing authority, issue date, expiry date (or "Does not expire")
   - Download document blob via `supabase.storage.from('certificate-documents').download(path)` using the existing `downloadAsBlob` helper pattern (extract path from documentUrl)
   - Detect file type from the URL extension:
     - **PDF**: Load with pdf.js `getDocument({ data: arrayBuffer })`, iterate pages, render each to canvas at scale 1.5, add as JPEG image to jsPDF (one page per jsPDF page)
     - **Images (jpg/jpeg/png)**: Load as Image element, add as full-page image scaled to fit within margins
     - **Other types**: Skip, increment skipped counter
   - Revoke blob URL immediately after processing
   - Running page count check: if total pages > 200, abort with descriptive error
6. **Footer on every page** (applied in final pass):
   - Center: `Page X of Y` (font 7, grey)
   - Right: `Generated by FlowSert` (font 7, grey)
7. Return `{ doc, included, skipped }`

**Key technical details:**
- Downloads use `supabase.storage.from('certificate-documents').download()` directly (respects RLS)
- Path extraction reuses the same pattern from `storageUtils.ts` (handles both full URLs and relative paths)
- pdf.js is already configured in the project (`pdfUtils.ts`)
- Canvas rendering for PDF pages uses scale 1.5 for balance of quality vs file size
- Each blob URL is revoked immediately after the document is processed

### 4. State and flow details for both dialogs

**New state variables:**
```ts
const [isBundleGenerating, setIsBundleGenerating] = useState(false);
const [bundleProgress, setBundleProgress] = useState<{ current: number; total: number; label: string } | null>(null);
```

**Download handler for bundle:**
```ts
const handleBundleDownload = async () => {
  const person = getSelectedPerson(); // single person
  if (!person) return;
  
  const certsWithDocs = person.certificates.filter(c => c.documentUrl);
  if (certsWithDocs.length > 30) {
    // Show confirmation dialog or toast
    if (!confirm(`This person has ${certsWithDocs.length} certificates. Continue?`)) return;
  }
  
  setIsBundleGenerating(true);
  try {
    const { doc, included, skipped } = await generateCertificateBundlePdf({
      person,
      companyName: businessName,
      projectName: selectedProject?.name,
      onProgress: (current, total, label) => setBundleProgress({ current, total, label }),
    });
    doc.save(`${person.name.replace(/[^a-z0-9]/gi, '_')}_certificate_bundle.pdf`);
    toast.success(`Certificate bundle downloaded. Included ${included} certificates${skipped > 0 ? `, skipped ${skipped} missing documents` : ''}.`);
  } catch (err) {
    toast.error(err instanceof Error ? err.message : 'Failed to generate certificate bundle');
  } finally {
    setIsBundleGenerating(false);
    setBundleProgress(null);
  }
};
```
