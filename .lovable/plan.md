
# Final Personnel Competence Matrix PDF Refinement

## Summary
Update `src/lib/competenceMatrixPdf.ts` to filter only active personnel, simplify metadata, add subtitle branding, and add right-aligned footer branding. Minor updates to call sites to remove `activePersonnelCount` (now redundant since the generator filters internally).

## Changes

### 1. `src/lib/competenceMatrixPdf.ts` -- Refine Generator

**Filter active personnel only:**
- At the start of the function, filter `personnel` to only include entries where `p.activated === true`
- All counts and rows derive from this filtered set

**Remove from interface:**
- Drop `activePersonnelCount` from `CompetenceMatrixOptions` (no longer needed -- all exported personnel are active by definition)

**Updated metadata block (4 lines only):**
```
Company: [name]
Project: [name or "Personnel Overview"]
Personnel in export: [count of rows in matrix]
Generated: [timestamp]
```
- Remove: Total Personnel, Active Personnel, Expiry warning, Location
- Left-aligned, font 8, muted grey

**Add subtitle below title:**
```
PERSONNEL COMPETENCE MATRIX        (font 16, bold, centered)
FlowSert Workforce Compliance      (font 9, normal, muted grey, centered)
```

**Updated footer (every page):**
- Center: `Page X of Y` (font 7, grey)
- Right-aligned: `Generated by FlowSert` (font 7, grey)
- No timestamps in footer

**Legend stays the same:**
```
Legend:  V -- Valid    E -- Expiring within 60 days    X -- Expired    - -- Not required / Not registered
```

**Header height adjustment:**
- Reduce header height slightly since we removed metadata lines (from 38 to ~34)

**Everything else stays:**
- Table columns (Name, Role, Type, Certificates), sorting, shading, grid lines, pagination, print safety

### 2. `src/components/ShareProjectDialog.tsx` -- Remove activePersonnelCount

Remove `activePersonnelCount` from the call to `generateCompetenceMatrixPdf` (line ~225).

### 3. `src/components/ExternalSharingDialog.tsx` -- Remove activePersonnelCount

Remove `activePersonnelCount` from the call to `generateCompetenceMatrixPdf` (line ~295).

## Technical Details

**Active filtering logic in generator:**
```ts
const activePersonnel = personnel.filter(p => p.activated);
```
All subsequent logic (sorting, cert collection, table body, counts) uses `activePersonnel` instead of raw `personnel`.

**Header draw function structure:**
```
y = 12: Title (centered, bold, font 16)
y += 7: Subtitle "FlowSert Workforce Compliance" (centered, font 9, grey)
y += 7: Company + Project (left-aligned, font 8, grey)
y += 4.5: Personnel in export + Generated
y += 4: Divider line
```

**Footer draw (every page):**
```ts
// Center
doc.text(`Page ${i} of ${pageCount}`, pageWidth / 2, pageHeight - 8, { align: 'center' });
// Right
doc.text('Generated by FlowSert', pageWidth - margin, pageHeight - 8, { align: 'right' });
```
