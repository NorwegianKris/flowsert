

## Align Project Card PDF with Existing Export Template

**Risk: GREEN** -- purely PDF styling changes, no data/schema/auth impact.

---

### Problem

The Project Card PDF export uses a different visual template than the Competence Matrix and Certificate Bundle exports. Key inconsistencies:

| Element | Project Card (current) | Matrix / Bundle (target) |
|---|---|---|
| Title | fontSize 20, mixed case | fontSize 14-16, UPPERCASE |
| Subtitle | None | "FlowSert Workforce Compliance" |
| Metadata | Sequential list, no alignment | Left/right split columns |
| Table theme | "striped" with blue headers `[59, 130, 246]` | "grid" with gray headers `[240, 240, 240]` |
| Table font | fontSize 9 | fontSize 7-8 |
| Footer | "Generated on..." centered, fontSize 8 | "Page X of Y" centered + "Generated by FlowSert" right, fontSize 7 |
| Divider | None | Thin gray line below metadata |

### Solution

Refactor the `generateProjectCardPdf` function inside `src/components/ShareProjectDialog.tsx` to match the shared template used by the other two exports.

### File Changed (1)

**`src/components/ShareProjectDialog.tsx`** -- rewrite the `generateProjectCardPdf` method:

1. **Header block**: Centered uppercase title "PROJECT CARD" at fontSize 14, followed by "FlowSert Workforce Compliance" subtitle at fontSize 8 in gray
2. **Metadata block**: Left column shows Project Name and Project Number; right column shows Duration and Generated timestamp. Uses fontSize 7, color `[128, 128, 128]`. Additional fields (Customer, Location, Project Manager, Work Category) listed below
3. **Divider**: Thin gray line (`[200, 200, 200]`, lineWidth 0.3) separating header from content
4. **Description**: If present, rendered below divider with bold label
5. **Personnel table**: Switch from `theme: 'striped'` with blue headers to `theme: 'grid'` with gray headers `fillColor: [240, 240, 240]`, `textColor: [0, 0, 0]`, fontSize 7. Grid lines use `[200, 200, 200]`
6. **Calendar table**: Same grid styling as personnel table
7. **Footer**: Replace single centered line with split layout -- "Page X of Y" centered and "Generated by FlowSert" right-aligned, both at fontSize 7 in `[160, 160, 160]`

### Result

All three exports (Project Card, Competence Matrix, Certificate Bundle) will share:
- Same title/subtitle layout with FlowSert branding
- Same metadata positioning pattern
- Same gray divider line
- Same table styling (grid theme, gray headers, consistent font sizes)
- Same footer format with page numbers and "Generated by FlowSert"
